Frontend: El Arte de la Interfaz del Usuario

HTML(HyperText Markup Language)

Es el lenguaje de marcado estándar utilizado para crear la estructura de las páginas web. Define elementos como encabezados, párrafos, enlaces e imágenes.

CSS (Cascading Style Sheets)

Es el lenguaje utilizado para describir la presentación de un documento escrito en HTML. Permite aplicar estilos, como colores, fuentes y layouts, para hacer las páginas web visualmente atractivas.

JavaScript

Es el lenguaje de programación que añade interactividad a las páginas web. Permite crear dinámicas como formularios interactivos, animaciones y actualizaciones en tiempo real sin recargar la página.
